# Install（安装）
在你的 React Native App 工程根目录下执行以下命令进行安装:

```
npm install --save react-native-row-text
```

# Usage（使用）
```

import FoldText from "react-native-row-text";

 <FoldText
	text={intro}
	maxLines={3}
/>


```
